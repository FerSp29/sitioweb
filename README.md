# Cronos Élite - Relojería de Lujo

¡Bienvenido a **Cronos Élite**! Este es un proyecto de desarrollo web que presenta un catálogo exclusivo de relojes de alta gama diseñado con una estética moderna, elegante y minimalista.

Este proyecto fue desarrollado como parte de mi formación en **Software Development** enfocándome en la creación de interfaces de usuario (UI) profesionales y una experiencia de navegación fluida.

---

## Características del Proyecto

- **Diseño Multi-página:** Navegación fluida entre la página de inicio y una página exclusiva para el catálogo.
- **Catálogo de Lujo:** Tarjetas de productos con efectos visuales avanzados, imágenes de fondo y degradados elegantes.
- **Interfaz Responsiva:** Diseño adaptado para dispositivos móviles, tablets y computadoras de escritorio.
- **Sección de Contacto:** Incluye un formulario funcional y un mapa interactivo (Google Maps) con estilo personalizado.
- **Animaciones CSS:** Efectos de elevación (hover), transiciones suaves y botones dinámicos.

---

## Tecnologías Utilizadas

Para la construcción de este sitio se utilizaron las siguientes tecnologías:

**HTML5** para la estructura semántica.
**CSS3** para el diseño, Grid, Flexbox y animaciones.
Tipografías **Playfair Display** (elegancia) y **Poppins** (modernidad).
