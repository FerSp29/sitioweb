⌚ Cronos Élite - Luxury Timepieces (V2.0)
¡Bienvenido a la versión optimizada de Cronos Élite! Este proyecto ha evolucionado de una landing page básica a una aplicación web multi-página con una arquitectura profesional, enfocada en el rendimiento y una estética minimalista de lujo.

---

🛠️ Evolución Técnica y Actualizaciones
En esta versión, el proyecto se centró en la escalabilidad y la limpieza visual:

🏗️ Arquitectura Modular (CSS)
He separado los estilos para mejorar la mantenibilidad del código:

  styles.css: Estilos globales (Navbar, Footer, Tipografía y Variables).
  nosotros.css: Ajustes de grid y equilibrio de contenido histórico.
  catalogo.css: Sistema de tarjetas (Cards) con efectos hover premium.
  contacto.css: Formulario con estilo "Urban" (diseño sin bordes, enfocado en líneas limpias).


🚀 Mejoras en la Interfaz (UI/UX)

  Navbar Pro: Rediseño del encabezado para una mejor jerarquía visual. Se aumentó la escala tipográfica y se optimizó el padding para un look más imponente.
  Sticky Header: Navegación siempre visible para mejorar la experiencia del usuario.
  Formulario Minimalista: Eliminación de contenedores pesados en la sección de contacto, usando ahora un diseño integrado al fondo con micro-interacciones en los campos.


---

🛠️ Tecnologías Utilizadas

  !HTML5 HTML5 Semántico - Uso de nav, section, header y footer.
  !CSS3 CSS3 Avanzado - Implementación de CSS Grid y Flexbox para layouts complejos.
  !Responsive Media Queries - Adaptabilidad total a dispositivos móviles.


---

✨ Características Destacadas

  Diseño Urban-Minimalist: Enfoque en espacios negativos y tipografía elegante (Playfair Display).
  Optimización de Espacios: Eliminación de márgenes colapsados y paddings innecesarios para un flujo de lectura natural.
  Navegación Fluida: Enlaces directos entre páginas con rutas relativas limpias.
  Mapa Interactivo: Integración de Google Maps con diseño adaptado al contenedor.
